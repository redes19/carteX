CarteX
projet final de l'UE dev web

Nous avons deux base de donné pour plus de sécurité : si l'une tombe, l'autre n'est pas affectée.

- Une BDD pour les cartes
- Une BDD pour les utilisateurs

Lancer le projet :
- git clone
- npm install
- npm start
- nodemon /src/Back/API/server.js

Ligne de commande

Pour le back js :
- npm install express --save
- npm install dotenv
- npm install cors
- npm install mariadb
- npm install bcrypt
- npm install jsonwebtoken

Pour le php :
- composer require --dev phpunit/phpunit


Niveau front :
- npm install react-router-dom
- npm install axios
- npm install react-slick --save
- npm install slick-carousel --save

style :
- npm install @mui/material @emotion/react @emotion/styled
- npm install @mui/icons-material

Lancer le server js
- nodemon server.js

Lancer server php:
- php -S localhost:8000
