CarteX

projet final de l'UE dev web

On a deux BDD pour + de sécurité : si l'une tombe, l'autre n'est pas affectée.

- Une BDD pour les cartes
- Une BDD pour les utilisateurs

Lancer le projet :

- git clone
- npm install
- npm start
